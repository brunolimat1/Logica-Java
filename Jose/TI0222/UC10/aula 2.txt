Back-up / restore (projeto Java - Eclipse)

Back-up:
1) Pasta workspace
2) Copiar a pasta inteira do projeto

Restore:
1) Copiar a pasta do projeto para o workspace
2) No eclipse:
   File -> Import
   Expandir a pasta General -> Existing projects into workspace
   Select root directory [browse] -> selecionar o workspace
   Selecionar o projeto a ser importado
